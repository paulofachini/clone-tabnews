# Processo de instalação

Pré-requisito:

- NodeJs
- Nvm

Executar abaixo o comando para instalar e definir a versão do NodeJS padrão do repositório, que pode ser verificada no arquivo `.nvmrc`.

```bash
nvm install
```

Instalação das dependências do projeto com base no `package.json`.

```bash
npm install
```
Executar o projeto.

```bash
npm rum dev
```
